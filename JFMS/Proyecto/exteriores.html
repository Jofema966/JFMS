Ubicación el corazón de nuestra comunidad, el parque de COVICOOP es el lugar ideal para relajarse y disfrutar de momentos únicos en familia o con amigos. Con amplias áreas verdes, senderos cubiertos de césped y flores para tomar fotografías este espacio brinda tranquilidad para todas las edades, ofrecemos un entorno perfecto para el descanso y la diversión.

Además, el parque cuenta con zona de picnic, canchas deportivas, y un parque infantil para que grandes y chicos puedan jugar como en casa.

Un lugar para eventos: Si estas buscando un lugar para realizar actividades familiares o celebraciones o eventos comunitarios nuestro parque es la opción.

No invitamos a disfrutar de este maravilloso lugar creado para usted y su familia!
